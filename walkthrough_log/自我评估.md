- **旧方案**：三段不同频率 beep（400/600/800Hz）区分下放/停顿/推起，认知负担高
- **新方案**：语音"推起/停/下放"标记阶段起点 + Tick 每秒填充持续时间

### 音频时间轴（以 1-1-3 节奏为例）

```
0s ──► 语音"推起" (~0.8s)
1s ──► 语音"停" (~0.5s)
2s ──► 语音"下放" (~0.8s)
3s ──► Tick
4s ──► Tick
(5s   = 离心3s结束)
```
