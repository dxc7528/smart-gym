## non-functional requirements

- 科学的力量训练逻辑是**“快起慢放”**（1-2秒向心，3-4秒离心）。在 Day 1 的动作中，极有必要对这两个阶段使用不同的时间控制。

核心逻辑与生理学事实：

肌肥大效应主要来自离心阶段（下放）：肌肉在拉长状态下对抗重力时，产生的肌肉微观撕裂（微损伤）最多。这是肌肉生长的核心信号。快速下放等同于放弃了 50% 以上的训练效益。

关节张力与安全性：侧平举若快速下放，肌肉张力会瞬间丢失，2kg 的重量加上重力加速度会直接由肩关节囊和韧带（尤其是冈上肌腱）承担，极易引发肩峰撞击和劳损。
